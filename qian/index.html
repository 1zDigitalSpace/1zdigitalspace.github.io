千的功能

群聊内使用千需要触发关键词

千 / 小千 / @千

1.大语言模型(llm)功能

触发关键词或私聊中发送任意非函数名内容即可触发llm功能,聊天信息中可以附带图片,可以查询系统时间和天气等信息,可以问一些简单题目

在千反应异常说梦话的时候请使用

new

2.运势检测打卡

运势 / 签到 / 打卡 - 发送生成的当天的运势和累计打卡天数

3.qq名片点赞

赞我

4.语音点歌

点歌 <歌名> - 根据序号点歌,可以附加歌手名,语音不完整

5.戳一戳互动

双击千的头像,触发一条互动消息

6.复读

自动触发

7.bilibili分享链接解析

检测到bilibili分享后自动发送解析的链接内容

8.github仓库解析

检测到github仓库链接后自动发送解析的链接内容

9.生成壁纸

moe

10.喜报悲报

喜报 <你的内容> 
悲报 <你的内容>

11.开盒

盒 <@好友>

12.bilibili直播提醒

仅在数码空间群内,特定直播间开播/下播时在群内发一条消息提醒

13.查Minecraft服务器状态

mcs <minecraft服务器地址>

14.早安晚安

早安 / 晚安

15.Arch Linux 软件包搜索

pkg <软件包名称> [仓库]

16.星空图解析

解析 <需要解析的图片>

17.站长工具

tcping <域名或IP地址> <port> - TCP端口连通性测试
ping <网址> - 测试指定网址的连通性
siteno <含http(s)的网址> - 测试指定网址的延迟
whois <域名> - 查询指定域名的whois信息
port <IP地址> - 扫描指定IP地址的端口
site <含http(s)的网址> - 获取网站截图
